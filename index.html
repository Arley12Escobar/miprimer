Acerca de mí

Hola, mi nombre es Arley Escobar. Soy un principiante de la programación y estoy empezando a estudiar programacion, para poder crear sitios web que sean visulamente atractivas.

Esta primer paguina es experimental, para tenerlo en cuenta, en este proceso aprendere JavaScript, CSS y HTML. Estos lenguajes nos ayudan para crar una base solida en el desarrollo web front-end





HTML es el lenguaje de marcado estándar utilizado para crear la estructura y el contenido de las páginas web. Se utiliza para definir la estructura y el significado de los elementos en una página web, como encabezados, párrafos, enlaces, imágenes, formularios, etc. HTML utiliza etiquetas y atributos para etiquetar y organizar el contenido, lo que permite a los navegadores web interpretar y mostrar correctamente la información.

JavaScript: JavaScript es un lenguaje de programación que se utiliza principalmente en el desarrollo web. A diferencia de HTML y CSS, que se centran en la estructura y la presentación de una página web, JavaScript se utiliza para agregar interactividad y funcionalidad dinámica a las páginas web. Con JavaScript, puedes controlar eventos, manipular elementos HTML, realizar cálculos, enviar y recibir datos del servidor, y mucho más. Es un lenguaje versátil que se ejecuta en el navegador del cliente.

CSS (Cascading Style Sheets): CSS es un lenguaje de hojas de estilo utilizado para controlar el aspecto y la presentación de las páginas web. Permite definir el estilo visual de los elementos HTML, como el color, la fuente, el tamaño, el espaciado, el diseño y otros atributos de presentación. CSS funciona separando la estructura del contenido (definido en HTML) y su presentación (definida en CSS), lo que facilita la creación de diseños coherentes y la personalización de la apariencia de una página web sin tener que modificar el contenido HTML subyacente.
